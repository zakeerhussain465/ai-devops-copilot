# AI DevOps Copilot

An AI-powered DevOps automation platform that generates deployment-ready infrastructure, CI/CD pipelines, Docker configurations, Kubernetes manifests, and cloud architecture recommendations from natural language prompts.

## Features

* AI-powered DevOps project generation
* Automated Dockerfile generation
* CI/CD pipeline generation
* Kubernetes manifest generation
* Infrastructure as Code templates
* Cloud deployment recommendations
* MERN application deployment support
* Architecture diagram generation

## Tech Stack

## Project Architecture

User → Frontend → Backend API → AI Engine → Generated DevOps Assets

Generated Assets:

* Dockerfiles
* Docker Compose
* Kubernetes YAML
* CI/CD Pipelines
* Infrastructure Templates
* Deployment Guides

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd ai-devops-copilot
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

## Environment Variables

Create a `.env` file inside backend:

```env
OPENAI_API_KEY=your_api_key
```
